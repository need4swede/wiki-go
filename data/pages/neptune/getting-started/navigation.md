# Navigation

Neptune uses a pill-shaped menu bar at the top of the screen for navigation, designed for quick access with the Apple TV remote.



## Tab Menu Bar

The menu bar appears at the top of every screen and contains the following tabs:

| Tab | Description |
|-----|-------------|
| **Settings** (gear icon) | Access app settings, profiles, and customization |
| **Home** | Your personalized home screen with all content |
| **Movies** | Browse your movie library |
| **Shows** | Browse your TV show library |
| **Discover** | Browse and request content via Jellyseerr |
| **Search** (magnifying glass) | Search your library and discover new content |

### Navigation Behavior

- **Focus to select**: Tabs are selected automatically when you focus on them - no need to press select
- **Smooth transitions**: Content updates seamlessly as you move between tabs
- **Focus memory**: When navigating back up to the menu bar from content below, focus returns to the currently selected tab



## Remote Controls

### Basic Navigation

| Action | Control |
|--------|---------|
| Move focus | Swipe on Touch Surface / D-pad |
| Select | Press Touch Surface / Select button |
| Go back | Menu button |
| Return to Home | Press and hold Menu button |

### Quick Actions

| Action | Control |
|--------|---------|
| Open Home Menu | Long-press any tab in the menu bar |
| Play/Pause (during playback) | Play/Pause button |
| Skip forward/back (during playback) | Swipe left/right on Touch Surface |



## Content Tabs

### Home Tab

The Home tab shows a personalized view of your library with multiple content rows. This is the default tab when you open Neptune.

**What you'll see:**
- Hero Spotlight carousel featuring content to watch
- Continue Watching row for resuming playback
- Next Up episodes from your shows
- New releases and recently added content
- Personalized recommendations

See [Home Screen](home-screen.md) for detailed information about customizing your home screen.

### Movies Tab

A filtered view showing only movie content from your library:
- Continue Watching (movies only)
- New Releases (movies only)
- Recently Added Movies
- Movie recommendations
- Because You Watched (movie suggestions)

### Shows Tab

A filtered view showing only TV content from your library:
- Continue Watching (episodes only)
- Next Up episodes
- New Releases (shows only)
- Recently Added Shows
- Show recommendations

### Discover Tab

Browse trending content and request new movies and TV shows. Requires Jellyseerr to be configured.

**What you'll see:**
- Trending Now - currently popular content
- Popular Movies - top-rated movies
- Popular TV Shows - top-rated series
- Coming Soon - upcoming releases

See [Discover](discover.md) for more information.

### Search Tab

Search across your library and Jellyseerr for content.

**Features:**
- Instant local search using FTS5 index
- Natural language queries (search by genre, actor, director)
- Jellyseerr integration to find and request new content

See [Search](search.md) for more information.



## Home Menu

Long-pressing any tab opens the Home Menu, which provides quick access to:

- **Section Customization** - Reorder or hide home screen sections
- **Edit Mode** - Enter edit mode for drag-and-drop section ordering
- **Settings shortcuts** - Quick access to common settings



## Tips

- **Navigate efficiently**: Swipe diagonally to move between the menu bar and content below
- **Quick tab switching**: While on the menu bar, swipe left/right to quickly move between tabs
- **Remember your place**: Neptune remembers your scroll position when switching tabs
