# Playback Settings

Configure video playback behavior.

---

## Trident Player

Neptune's custom video engine for improved compatibility.

| Setting | Description |
|---------|-------------|
| **Enabled** | Use Trident for all playback |
| **Disabled** | Use standard AVPlayer |

**Note:** Trident is experimental. Disable if you experience issues.

See [Trident Player](/playback/trident-player) for technical details.
