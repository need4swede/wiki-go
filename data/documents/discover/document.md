# Discover

Browse trending content and request movies and shows. Requires Jellyseerr.



## Content Sections

| Section | Description |
|---------|-------------|
| **Trending Now** | Currently popular content |
| **Popular Movies** | Top-rated films |
| **Popular TV Shows** | Top-rated series |
| **Coming Soon** | Upcoming releases |



## Status Badges

| Badge | Color | Meaning |
|-------|-------|---------|
| Available | Green | In your library |
| Requested | Yellow | Pending approval |
| Processing | Blue | Being downloaded |
| Requestable | Accent | Can be requested |



## Requesting Content

**Movies:** Select item, press Request, sent to Radarr.

**TV Shows:** Select item, press Request, choose seasons, sent to Sonarr.



## Admin Features

Administrators see a **Pending Requests** banner when approvals are needed.

- Review each request
- Approve or decline



## Setup

If Jellyseerr isn't connected:

1. Select **Connect to Jellyseerr**
2. Enter server URL
3. Sign in with Jellyfin credentials

Your session is saved for future use.
