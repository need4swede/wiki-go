# Themes

Customize Neptune's visual appearance.



## In This Section

| Page | Description |
|------|-------------|
| [Built-in Themes](/personalization/themes/built-in) | Six pre-designed themes |
| [Custom Themes](/personalization/themes/custom) | Import custom JSON themes |



## What Themes Control

- Background colors and animations
- Text colors and opacity
- Accent colors for buttons
- Focus indicators and borders
- Player controls appearance
- Media quality badges



## Changing Themes

**During setup:** Choose a theme as the final step.

**From Settings:**

1. Navigate to **Settings** tab
2. Browse available themes
3. Focus on a theme to preview live
4. Select to apply



## Per-Profile Themes

Each user profile can have its own theme. Neptune switches themes automatically when you switch profiles.



## Tips

- **Preview first**: Focus on themes to see them live
- **OLED displays**: Use Void theme for true blacks
- **Dark rooms**: Ocean or Midnight work best
