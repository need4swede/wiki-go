# Item Details

When you select content, Neptune displays detailed information and actions.



## In This Section

| Page | Description |
|------|-------------|
| [Movies](/browsing/item-details/movies) | Movie pages, quality badges, cast |
| [TV Shows](/browsing/item-details/tv-shows) | Series, seasons, episodes |
| [Extras](/browsing/item-details/extras) | Special features, trailers, behind-the-scenes |



## Common Elements

All detail views display:

- **Backdrop**: Full-screen background image
- **Logo/Title**: Official logo or text
- **Metadata**: Year, runtime, rating, genres
- **Overview**: Plot summary
- **Play button**: Start or resume playback



## Menu Options

Press the **Menu button** for actions:

| Action | Description |
|--------|-------------|
| Add/Remove Favorite | Toggle favorite status |
| Mark Watched/Unwatched | Toggle played status |
| Refresh Metadata | Update from server |

Admin users see additional options for image management and advanced refresh.
