## Key Features

- **Hero Spotlight**: Full-bleed backdrop with parallax scrolling, showcasing what to watch next
- **Trident Player**: Custom video engine with HDR, Atmos, and codec support
- **Semantic Search**: AI-powered thematic search across your library
- **Jellyseerr Integration**: Browse and request new content
- **Custom Themes**: Six built-in themes plus JSON import for custom designs



## Requirements

- **Apple TV**: tvOS 17.0+
- **Jellyfin Server**: Any recent version
- **Optional**: Jellyseerr for content discovery
- **Optional**: LLM provider for prompt-based search features
