# Jellyseerr Setup

Connect to Jellyseerr for content discovery and requests.



## What is Jellyseerr?

Jellyseerr lets you:

- Browse trending and popular content
- Request movies and TV shows
- Track request status



## Automatic Detection

Neptune tries to find Jellyseerr automatically:

- **IP addresses:** Checks port 5055 (e.g., `192.168.1.100:5055`)
- **Domains:** Checks for `jellyseerr` subdomain

If detected, select **Login to Jellyseerr**.



## Manual Setup

**Subdomain setup:**

1. Select **Yes** when asked about subdomain hosting
2. Enter the subdomain name (e.g., `jellyseerr`)
3. Press **Connect**

**Custom URL:**

1. Select **No, enter full URL**
2. Enter your complete Jellyseerr address
3. Press **Connect**



## Signing In

Neptune uses your Jellyfin credentials:

1. Username is pre-filled
2. Enter your Jellyfin password
3. Press **Sign In**

Your session is saved for future use.



## Skipping Setup

Select **Skip for Now** if you don't have Jellyseerr. Configure it later in **Settings**.
